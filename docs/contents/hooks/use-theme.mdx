---
title: useTheme
description: >-
  `useTheme` is a custom hook that returns a function for retrieving and
  switching themes.
package_name: "@yamada-ui/core"
with_description: true
---

## Import

```ts
import { useTheme } from "@yamada-ui/react"
```

## Usage

```tsx functional=true
const { theme } = useTheme()

return <Text lineClamp={3}>{JSON.stringify(theme)}</Text>
```

:::note
If you want to learn more about the theme, please check [here](/styled-system/theming).
:::

### Switching Themes

```tsx functional=true
const { themeScheme, changeThemeScheme } = useTheme()

return (
  <>
    <Text>The current scheme is "{themeScheme}"</Text>

    <Wrap mt="md" gap="md">
      <For
        each={[
          { changeThemeScheme: "base", children: "Base Theme" },
          {
            colorScheme: "pink",
            changeThemeScheme: "pink",
            children: <>Pink Theme</>,
          },
          {
            colorScheme: "purple",
            changeThemeScheme: "purple",
            children: <>Purple Theme</>,
          },
          {
            colorScheme: "green",
            changeThemeScheme: "green",
            children: <>Green Theme</>,
          },
        ]}
      >
        {(item, index) => <Button key={index} {...item} />}
      </For>
    </Wrap>
  </>
)
```

:::note status=warning
In order to switch themes, you need to prepare multiple themes. For more details, please check [here](/styled-system/theming/switch-themes).
:::
