## インポート

```ts
import { useTheme } from "@yamada-ui/react"
```

## 使い方

```tsx functional=true
const { theme } = useTheme()

return <Text lineClamp={3}>{JSON.stringify(theme)}</Text>
```

:::note
テーマをもっと学びたい場合は、[こちら](/styled-system/theming)をご覧ください。
:::

### テーマを切り替える

```tsx functional=true
const { themeScheme, changeThemeScheme } = useTheme()

return (
  <>
    <Text>The current scheme is "{themeScheme}"</Text>

    <Wrap mt="md" gap="md">
      <For
        each={[
          { changeScheme: "base", children: "Base Theme" },
          {
            colorScheme: "pink",
            changeScheme: "pink",
            children: <>Pink Theme</>,
          },
          {
            colorScheme: "purple",
            changeScheme: "purple",
            children: <>Purple Theme</>,
          },
          {
            colorScheme: "green",
            changeScheme: "green",
            children: <>Green Theme</>,
          },
        ]}
      >
        {({ colorScheme, changeScheme, children }, index) => (
          <Button
            key={index}
            colorScheme={colorScheme}
            onClick={() => changeThemeScheme(changeScheme)}
          >
            {children}
          </Button>
        )}
      </For>
    </Wrap>
  </>
)
```

:::note status=warning
テーマを切り替えるには、複数のテーマを用意する必要があります。詳しくは、[こちら](/styled-system/theming/switch-themes)をご覧ください。
:::
